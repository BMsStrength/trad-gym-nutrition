# TRADジム栄養指導アプリ — 公開手順書

## 必要なもの
- メールアドレス（Anthropic・Supabase・Vercelの登録用）
- クレジットカード（Anthropic APIの利用料金 ※登録だけは無料）
- スマホかPC

---

## STEP 1: Anthropic APIキーを取得

1. https://console.anthropic.com にアクセス
2. 「Sign Up」でアカウント作成
3. クレジットカードを登録（使った分だけ請求、登録だけなら無料）
4. 左メニュー「API Keys」→「Create Key」
5. 表示されたキー（sk-ant-...）をコピーして保管

💰 料金目安：1回の食事分析 約1〜3円
　　　　　　 会員50人 × 1日3食 × 30日 ≒ 月4,500〜13,500円

---

## STEP 2: Supabaseの設定（会員データ保存用）

1. https://supabase.com にアクセス
2. 「Start your project」でアカウント作成（GitHub or メール）
3. 「New Project」を作成
   - Organization: 任意
   - Name: trad-gym
   - Database Password: 安全なパスワードを設定（保管すること）
   - Region: Northeast Asia (Tokyo) を選択
4. 作成完了後（約1分）、左メニュー「SQL Editor」を開く
5. supabase_setup.sql の内容を貼り付けて「Run」を実行
6. 左メニュー「Project Settings」→「API」で以下をコピー
   - Project URL → NEXT_PUBLIC_SUPABASE_URL
   - anon / public → NEXT_PUBLIC_SUPABASE_ANON_KEY
   - service_role → SUPABASE_SERVICE_ROLE_KEY（秘密にすること）
7. 左メニュー「Authentication」→「URL Configuration」
   - Site URL: https://あなたのVercelURL.vercel.app を設定（STEP3後に設定）

---

## STEP 3: GitHubにコードをアップロード

1. https://github.com にアカウント作成（なければ）
2. 「New repository」→ 名前: trad-gym-nutrition → Create
3. このフォルダのファイルを全てアップロード
   （「uploading an existing file」から一括アップロード可）
4. .env.local.example はアップロードしてOK（実際のキーは含まない）

---

## STEP 4: Vercelにデプロイ（公開URL発行）

1. https://vercel.com にアクセス（GitHubアカウントでログイン）
2. 「Add New Project」→ GitHubのtrad-gym-nutritionを選択
3. 「Environment Variables」に以下を追加：

   | キー名 | 値 |
   |--------|-----|
   | ANTHROPIC_API_KEY | sk-ant-... |
   | NEXT_PUBLIC_SUPABASE_URL | https://xxxxx.supabase.co |
   | NEXT_PUBLIC_SUPABASE_ANON_KEY | eyJ... |
   | SUPABASE_SERVICE_ROLE_KEY | eyJ... |

4. 「Deploy」をクリック
5. 数分後にURLが発行される（例: trad-gym-nutrition.vercel.app）

---

## STEP 5: Supabaseにサイトを登録（メール確認リンクを有効化）

1. Supabase「Authentication」→「URL Configuration」
2. Site URL に Vercel の URL を入力して保存

---

## STEP 6: 会員さんに共有

- URLをLINEやQRコードで共有するだけでOK
- スマホでURLを開いて「ホーム画面に追加」でアプリのように使える
- iOS: Safari で開いて「共有」→「ホーム画面に追加」
- Android: Chrome で開いて「⋮」→「ホーム画面に追加」

---

## トレーナー向け管理

Supabase の「Table Editor」→「meal_records」から全会員の記録を確認できます。
（将来的に専用の管理画面も追加可能）

## よくある質問

Q: アップデートしたい場合は？
A: GitHubのファイルを更新すると自動でVercelに反映されます。

Q: 会員数が増えてきたら？
A: Vercel・Supabaseとも無料枠で100人程度まで対応可能です。

Q: セキュリティは大丈夫？
A: Supabase RLSにより、会員は自分のデータのみアクセス可能です。
